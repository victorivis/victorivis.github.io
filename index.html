Série de Fourier

A série de fourier é uma expressão matemática desenvolvida por Jean Baptiste Joseph Fourier que tem aplicações no estudo de ondas sejam elas na música, em trocas de calor ou até mesmo frequência da luz. A equação pode ser expressa por: f ( x ) = a + ∑ n = 1 ∞ [ a n cos ( 2πnx / T ) + b n sin ( 2πnx / T ) ]
